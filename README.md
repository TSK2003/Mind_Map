# Mind Map

Mind Map is a Windows-first, offline-first personal knowledge operating system that combines visual mind maps, structured notes, backlinks, media storage, project planning, and AI-assisted recall.

This repository contains the production-grade starter scaffold:

- Electron + React + TypeScript desktop shell.
- React Flow mind map canvas.
- Tiptap rich editor.
- Knowledge graph and productivity dashboard surfaces.
- Local `.brain` open/save bridge.
- SQLite schema for the production vault engine.
- Windows NSIS packaging plan.

## Run

```powershell
npm install
npm run dev
```

## Build

```powershell
npm run build
npm run package:win
```
